# C++ OOP Contact Book System

## Project Overview
This is a Contact Book Management System developed in C++ using Object-Oriented Programming (OOP) concepts (Spring 2024).  
The application is designed to manage personal and professional contacts efficiently, providing a structured and user-friendly console-based interface. It also ensures data persistence through file handling.

## Features
- Add, delete, and update contacts  
- Search contacts using name, number, email, or address  
- Sort contacts by first name or last name  
- Merge duplicate contacts automatically  
- Maintain search history with date and time tracking  
- Create and manage groups (add/remove members)  
- Store and load data using file handling  
- Manage contact list size dynamically  

## Group Feature (Short Info)
The system allows users to organize contacts into groups for better management. Users can create multiple groups, add contacts to specific groups, view group members, remove members, and delete groups when needed. Group data is also stored in files for persistence.

## How to Run
1. Open the project in any C++ IDE (e.g., Visual Studio or CodeBlocks)  
2. Ensure all header (.h) and source (.cpp) files are properly included  
3. Compile the project  
4. Run the main.cpp file  
5. Enter the contact list size and follow the menu options displayed on the screen
